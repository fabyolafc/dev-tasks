# 🎯 Frontend – Dev Tasks

Aplicação frontend para gerenciamento de tarefas, desenvolvida para consumir uma **API Node.js**.  
O projeto permite criar, listar, atualizar e remover tarefas de forma simples e intuitiva.

🔗 **Frontend:** https://dev-taskss.vercel.app  
🔗 **Backend:** https://api-taskss.onrender.com  
📄 **Swagger:** https://api-taskss.onrender.com/api-docs

## 📌 Funcionalidades

- Listar tarefas cadastradas
- Criar novas tarefas
- Atualizar tarefas 
- Excluir tarefas
- Filtro de tarefas
- Integração com API REST
- Feedback visual com Toastify

---

## 🖥️ Tecnologias Utilizadas

- **HTML5** – Estrutura da aplicação
- **CSS3** – Estilização da interface
- **JavaScript (Vanilla JS)** – Lógica e interações
- **Fetch API** – Consumo da API backend
- **Toastify.js** – Notificações visuais
- **Vercel** – Deploy do frontend
